ABOUT GAMEPADLA  
The gamepadla project was created to test the latency of game controllers. It is open source and allows any enthusiast to measure the latency of gamepads with the help of improvised tools in an Arduino-based device.
Watch the video for more detailed instructions on how it works https://www.youtube.com/channel/UCjBKBos7dBDO6au6Zn3TXNg
